## About

About たとえば

たとえば (or 例たとえば in its kanji form) is an adverb in Japanese that is used at the beginning of a sentence, or an individual statement, in order to present an example. It can be translated almost directly as 'for example'.

たとえば、ドイツとかは？

For example, how about Germany, or something?

たとえば、冷つめたい物ものを食たべたときに歯はが痛いたいです。

For example, my teeth hurt when I eat cold things.

Fun Fact

例たとえば should primarily be used when the 'example' that is being given is as simple as possible. If you want to explain something in further (in-depth) detail, 詳くわしく 'in full' would be used as the adverb instead.

詳くわしく言いうと、トマトのように種たねがいっぱいある食たべ物ものが嫌きらいです。

More specifically, I don't like foods that have a lot of seeds, like tomatoes.

たとえば、「英語えいごが好すきですか。」と聞きいてみたら？

For example, why don't you ask 'Do you like English?'

たとえば、翻訳者ほんやくしゃになったら日本にほんに住すむことが出来できる。

For example, if you become a translator, you will be able to live in Japan.

漢字かんじを覚おぼえるサイトがたくさんあります。たとえば、ワニカニだ。

There are a number of sites you can use to learn kanji. For example, Wanikani.

たとえば毎日まいにち日本語にほんごを話はなすなら、日本語にほんごのレベルが上あがる。

For example, if you speak Japanese every day, your Japanese level will rise.

私わたしは、たとえばあそこにあるイタリアンレストランのような店みせを作つくりたい。

For example, I want to make a restaurant like that Italian restaurant over there.

色々いろいろな国くにに行いって見みたい。たとえば、ベトナム。

I want to go to a variety of countries. Vietnam for example.

子供こどものころ何なんでも食たべることが出来できた。たとえば、納豆なっとうが好すきだった。

When I was a child, I could eat anything. I liked natto for example.

たとえば、毎日まいにち日本語にほんごを話はなしたら、だんだん上手じょうずになってくるよ。

For example, if you speak Japanese every day, you will gradually get better.

たとえば、違ちがう国くにの歴史れきしを勉強べんきょうしたら文化ぶんかの違ちがいが分わかってくる。

For example, if you study the history of other countries, you will come to realize differences in culture.

いつも丁寧ていねいに話はなす方ほうがいいよ。たとえば、「ありがとう」と「すみません」は必要ひつような言葉ことば。

It is better to always speak politely. For example, 'thank you' and 'excuse me' are necessary phrases.

たとえば、「私わたしはお金かね持もち」って子供こどもに言いったら、子供こどもたちが色々いろいろなものを買かってもらいたくなるだろう。

For example, if you say that you are rich, your children will want to have you buy them various things.

この授業じゅぎょうは本当ほんとうに大変たいへんだ。たとえば、明日あしたまでにこの文章ぶんしょうを書かき終おわらなかったら、点数てんすうがゼロになる。

This class is really giving me trouble. For example, if I don't finish writing this essay by tomorrow, I'll get zero points.
