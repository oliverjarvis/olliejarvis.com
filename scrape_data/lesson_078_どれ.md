## About

About どれ

どれ is from the same family of words as これ, それ, and あれ. These words are called こそあど言葉ことば in Japanese. Unlike the other 3, which all point to a specific thing, words starting with ど are designed for enquiring about something unknown. どれ can roughly be translated as 'which' in English, and is used for enquiring about a group (of 3 or more things).

どれが飛とびますか。

Which one of these flies?

どれが新あたらしいですか？

Which one is new?

どれが綺麗きれいですか？

Which one is beautiful?

どれが弟おとうとですか？

Which one is your younger brother?

From these examples, we can see that the particle が is often used with どれ, especially when preceding questions. However, when どれ follows a noun, は will be used.

あなたの車くるまはどれ？

Which one is your car?

When there are only two choices to select between, どちら + の, or どっち + の will be used instead of どれ. This is a habit that has become part of the language over time, and was not originally how these words were used.

どちらのケーキを食たべますか？

Which cake would you like to eat? (Out of these two choices)

どっちのテストが難むずかしい？

Which test is hard? (Out of these two choices)
